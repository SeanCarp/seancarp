Projects Showcase

CarpCade Arcade Cabinet

A high school project building a custom arcade cabinet for my father. I designed it to his specific elbow and eye height, sketching it by hand, creating a 1:12 scale model, and then a 1:1 scale profile. I integrated a Raspberry Pi, old monitor, speakers, and arcade controls. This project taught me a lot about woodworking, design for manufacturing, and repairability.

GARY (General Artificial Response Yapper)

An attempt at a more efficient AI assistant. Instead of relying on a single large LLM, I am splitting the logic to integrate my own scripts and APIs. It uses an NLP pipeline (intent classifier, NLU, NER) to interpret input, run local scripts, or save reminders. A final, style-focused transformer mimics the user's attitude. The entire system is designed to run on a small, portable computer (LeRengade) for a challenge in efficiency.

LAC (Lunar Autonomy Challenge)

As project lead for the AIWVU club, I worked on this NASA/Johns Hopkins APL competition to create an autonomous system for navigating simulated lunar terrain. This was my first large collaborative coding project, using ROS to interface between a Python simulator and C++ (ORB-SLAMV3). I also solved complex networking challenges, setting up VLANs and VcXsrv to allow multiple users to access the single simulator laptop. We were able to place 5th out 31 teams nationally!

WVU BAJA Data Acquisition (DAQ)

Volunteered for the BAJA DAQ sub-team to collect sensor data on the off-road buggy. The system displays crucial information (RPM, brake pressure, etc.) to the driver via an LCD screen and is designed to alert the pit team of breakdowns. This involved hardware integration, 3D printed enclosures, and microcontroller programming.

Claw Machine Design

An ongoing personal project to design and build a full-size claw machine. This project highlights my ability to take an idea from a napkin sketch to a detailed 3D CAD model. It's a massive time commitment I'm chipping away at, allowing me to be creative and iterate on complex design solutions.

Raspberry Pi Motor Control

A self-taught high school project to learn basic electronics. I wired an old drone motor in a 3D printed stand to a transistor, controlling it with a PWM pin on a Raspberry Pi. The motor speed was based on the input of a joystick, converted from analog to digital with an ADC converter. This project built a strong foundation in electronics that proved invaluable in college.
